(1) 정수형

(2) 실수형    
**지수표현방식
1e9는 10의 9제곱과 같다.  
예를 들어, 최단 경로로 가능한 최댓값이 10억 미만이라면 무한(INF)을 1e9로 표현할 수 있다.  

1e4 # 10000  
75.25e1 # 752.5  
3954e-3 # 3.954

(4) round 함수  
첫 번째 인자 - 실수형 데이터  
두 번쨰 인자 - (반올림하고자 하는 위치 - 1)  

ex) 123.456을 소수점 셋째 자리에서 반올림하려면, round(123.456, 2)  
결과 # 123.46
